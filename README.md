# 🎥 YouTube Playlist Transcript Extractor

A Python script that extracts complete information from every video in a YouTube playlist, including transcripts, metadata, and multiple export formats.

---

# ✨ Features

✅ Accepts a YouTube Playlist URL

✅ Automatically fetches all videos in the playlist

✅ Extracts metadata for every video

- Video Title
- Video URL
- Channel Name
- Video Duration
- Publish Date (when available)
- Complete Transcript

✅ Combines all extracted information into a single structured text document

✅ Supports exporting data into multiple formats

- TXT
- CSV
- JSON
- Markdown (MD)



# 📦 Requirements

- Python 3.10+
- yt-dlp
- youtube-transcript-api
- pandas
